# Turno da Meia-noite (Midnight Shift)

**Turno da Meia-noite** é um jogo de puzzle atmosférico onde o jogador controla um personagem em um cemitério durante a noite. O objetivo é empurrar as lápides para suas posições corretas antes que os mortos levantem a meia-noite, criando uma experiência de puzzle com elementos de urgência temporal.

Título: Turno da Meia-noite
Gênero: Puzzle / Sokoban temático
Plataforma: PICO-8 (8-bit, 128x128px, 16 cores)
Duração: 5–10 minutos por jogador casual (ideal para gamejam)
Loop Principal: Empurre lápides até suas posições corretas antes da meia-noite.

## Como Executar

### Pré-requisitos
- PICO-8 instalado no sistema

### Executando o jogo
Use o comando `make run`

Ou simplesmente abra o arquivo `midnightshift.p8` diretamente no PICO-8.
